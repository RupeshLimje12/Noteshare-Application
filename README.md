# 📚 Noteshare-Application

A full-stack web application that allows students and learners to upload, share, explore, and interact with educational notes. The platform provides features such as user authentication, note publishing, commenting, profile management, and real-time messaging, creating a collaborative learning environment.

## 🚀 Features

- User Registration and Login Authentication
- Upload and Share Study Notes
- View and Explore Notes from Other Users
- Like and Comment on Notes
- User Profile Management
- Search Users Functionality
- Real-Time Chat/Messaging System
- Responsive User Interface
- Secure Backend APIs

## 🛠️ Technology Stack

### Frontend
- React.js
- React Router
- Redux Toolkit
- Material UI
- Styled Components
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Socket.io

## 📂 Project Structure

```bash
NoteSharing/
│
├── frontend/      # React Frontend
├── backend/       # Node.js & Express Backend
└── database/      # Database Configuration
```

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/ankitkr437/NoteSharing.git
cd NoteSharing
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

## 🔑 Environment Variables

Create a `.env` file in both frontend and backend directories and configure the required environment variables such as:

```env
MONGO_URL=your_mongodb_connection_string
PORT=5000
```

## 🎯 Purpose

The main objective of this project is to provide a centralized platform where students can easily share, access, and collaborate on study materials, improving knowledge exchange and learning efficiency.
